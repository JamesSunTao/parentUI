#parentui-row

## Attributes
|  参数 | 说明 | 类型  | 可选值  | 默认值  |
| ------------ | ------------ | ------------ | ------------ | ------------ |
| tag | 自定义元素标签 | string| - | div |
| gutter | 列元素之间的间距（单位为 px） | number,string | - | - |
| type | 布局方式 | string | flex | - |
| justify | Flex 主轴对齐方式 | string | end center space-around space-between | start |
| align | Flex 交叉轴对齐方式 | string | center bottom top | top |
